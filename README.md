
# Grocery Price Tracker (n8n)

This workflow tracks grocery prices from Pak'nSave, New World, and Woolworths in New Zealand using SerpAPI and Google Sheets.

## Features
- Reads grocery items from Google Sheets
- Searches prices via SerpAPI
- Filters results by supermarket domain
- Uses AI Agent to extract lowest prices
- Updates Google Sheet with structured data

## Technologies
- n8n
- SerpAPI
- Google Sheets
- AI Agent
